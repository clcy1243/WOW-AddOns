ImprovedTotemFrame
==================

* by Phanx <addons@phanx.net>
* http://www.wowinterface.com/downloads/info-ImprovedTotemFrame.html
* http://wow.curse.com/downloads/wow-addons/details/improvedtotemframe.aspx


Description
-----------

ImprovedTotemFrame modifies the totem frame in the default UI.

* Totem frame is movable. Type “/movetotems” to toggle the locked state.
* Totem frame is hidden when you are dead, flying, mounted, possessing something, or in a vehicle.
* Totem buttons have text timers for totem durations.
* Totem buttons can be shift-right-clicked to destroy the associated totem.
* Summon and Recall buttons are hidden.
